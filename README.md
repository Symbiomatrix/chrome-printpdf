# chrome-printpdf
Print page to pdf.

## Features
- Press extension button to create and download pdf.
- Automatic layout: Landscape without margins.
- Multi page if the page is not lazy loaded.
- Captures most media including videos, but notably youtube players show as empty.

## Limits
- Uses debug interface which is presumably insecure, might break.
- Cannot install the .crx versions on recent brave / chrome, since it's not from the web store.
Dev mode -> load unpacked -> point to folder with js files works.
